
Overhauled by ZeroNexus (With help from Purple Nebula)

In honor of the original Hiigaran Descendants mod, this mod wishes to revive the mod and expand it, hoping to eventually add ships from HW1,HW2 and HW3 (very hopefully).

Version 1.0 (Reborn Release)

-Updated Outdated Code to bring it inline with the latest version.
-Added Hard Dependency on LazyLib, GraphicsLib, MagicLib (It was missing from base version, causing issues).
-Added UAF Relationship if United Aurora Federation is installed (Future updates will change more when installed together).

Version 1.1 (Quantum Teleportation Update) (Modder Note: Don't fiddle with teleporters unless you want to clone yourself!)

-Updated some ingame text during run setup (Relating to the Hiigarans).
-Updated the various interactions the player has with Hiigaran fleets at the various relationship levels.
-Tweaked and modified the Hiigarans relationships with the rest of the sector (Vanilla and some modded, more to change in future)
-Added the Sajuukhar as a super ship start.
-Teleported Polaris to the southern sector, to be nearby but not close neighbours to modded factions like the UAF, New Batavia Republic and others.
-Hiigarans have established Democracy.
